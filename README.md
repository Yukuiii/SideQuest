## 📖 简介 (Introduction)

欢迎使用 **Side Quest** —— 专为现代 **Vibe Coder** 打造的终极“生产力”（或者反生产力？）伴侣。

我们需要面对现实：你写好了 Prompt，发给了 AI，然后……就是等待。等待 Token 一个个蹦出来，等待测试通过，等待构建完成。

**既然都在等，为什么要盯着那个转圈的 Loading 图标发呆？不如来做个“支线任务”吧！**

**Side Quest** 是一款 VS Code 插件，旨在填补你编程流中的那些碎片时间。无论是追更你最爱的小说、扫一眼最新的科技圈瓜，还是单纯地放空大脑，我们都让你**无需离开编辑器**即可完成。

> “以前等 Copilot 写代码的时候我会切出去刷网页。现在我直接开启 Side Quest。老板以为我在查阅日志。” —— _某匿名 10x 工程师_

## ✨ 功能特性 (Features) and so on...

### 📚 阅读者 (Novel Mode) _[开发中]_

- **沉浸式阅读**：支持直接在侧边栏或底部面板加载并阅读 `.txt` 或 `.epub` 本地文件。
- **隐蔽模式 (Stealth Mode)**：
  - 字体和配色完美融入代码主题，看起来就像是在 Debug。
  - **老板键**：一键隐藏小说内容，瞬间切换回伪造的 Stack Trace（开发中 ™）。
- **自动书签**：自动记录阅读进度，就算 VS Code 崩溃了，你的修仙进度也不会丢。

### 📰 情报员 (News Mode) _[开发中]_

- **热点聚合**：在等待编译时，快速浏览 Hacker News、掘金或知乎热榜。
- **摸鱼也要摸硬核的**：只看标题，绝不沉迷，保持 Vibe。

### 📈 操盘手 (Market Mode) _[开发中]_

- **盯盘助手**：在状态栏实时显示股票或 Crypto 价格。
- 绿了就继续 Coding，红了就加倍 Coding。

## 🔮 未来计划 (Roadmap)

我们要把 Vibe 进行到底：

- [ ] **🎵 氛围组 (Vibe Station)**: 内置 Lo-Fi 电台与白噪音，Coding 也要有 BGM。

  - **Lo-Fi Radio**：内置播放器，一键播放 "Coding Mode" 背景音。
  - **白噪音生成器**：提供雨声、键盘敲击声或咖啡馆背景音，助你进入心流。
  - **社死保护**：默认静音启动，绝不外放。

- [ ] **🕹️ 街机厅 (Arcade)**: 在 Output 面板玩 ASCII 版贪吃蛇与 2048。

  - **ASCII 游戏**：无需图形界面，直接在日志输出窗口玩贪吃蛇、俄罗斯方块。
  - **摸鱼竞速**：在等待部署的时候，测测你的打字速度 (WPM)，顺便练练 Vim 快捷键。

- [ ] **🧠 碎片充能 (Brain Buff)**: 随机推送 LeetCode 思路或英语单词，等待时间也能变强。

  - **LeetCode "每日一题" (Lite 版)**：只看题目和思路，不写代码。保持大脑活跃，假装在思考算法。
  - **单词卡片**：每写 100 行代码，强制弹出一个 GRE 单词，通过了才能继续写。

- [ ] **🎭 影帝模式 (Oscar Mode)**: 一键生成“看起来很复杂”的虚假终端日志，用来应付突然出现的产品经理。
  - **伪造编译 (Fake Build)**：点击紧急按钮，终端立刻开始疯狂输出看起来很厉害的日志（其实是乱码），进度条永远卡在 99%。
  - **黑客打字机**：自动在屏幕上输入一段复杂的 Linux 内核代码，让你看起来像是个正在拯救服务器的黑客。

## 🚀 快速开始 (Getting Started)

1.  从 VS Code 插件市场安装本插件。
2.  打开命令面板 (`Cmd+Shift+P` / `Ctrl+Shift+P`)。
3.  输入 `Side Quest: Start` 开启你的冒险。
4.  选择 `Load File` 加载你本地的“学习资料”。
5.  **Vibe out.**
